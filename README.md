# Spanilaza
Open source Spanish learning software that uses a custom built API on Django RF.

Included modes:

- Conjugations: 100 most common verbs
- Conjugations: Haber
- 250 Most common nouns
- 250 Most common adjectives
- Por vs Para
- Subjunctive practice

...and several more!
